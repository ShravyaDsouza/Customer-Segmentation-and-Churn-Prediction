# Customer Segmentation & Churn Prediction

## Overview
This project applies machine learning techniques to the **Online Retail II** dataset to perform customer segmentation and churn prediction. We use K-Means for unsupervised learning to identify customer segments and K-Nearest Neighbors (KNN) for predicting customer churn.
